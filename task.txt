Тестове завдання back-end developer
1) Потрібно зробити API сервіс на Express, можна без фронтенд частини.
Сервіс повинен мати авторизацію по JWT.
Логін/пароль можуть бути фіксованими.
При успішній авторизації повинен повертати access token, який має використовуватися та валідуватися при всіх подальших запитах.
2) При надсиланні на API csv-файлу з невеликою таблицею (дані на свій вибір) потрібно вставляти її в MongoDB (хмара). Попередні дані очищати. Використовуйте Mongoose.
3) Третій ендпоінт повинен віддавати ці дані до JSON.
